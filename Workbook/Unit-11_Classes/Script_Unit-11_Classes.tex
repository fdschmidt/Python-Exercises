\subsubsection*{Exercise 11.03 \red{[M]}}
A complex number can be expressed as
\begin{center}
	$a + i \cdot b$
\end{center}
where $a$ and $b$ are real numbers and $i$ is the imaginary number which solves $i^2 = -1$.\\
Generate a class {\code{Complex}}. The class contains the attributes {\code{real}} and {\code{imaginary}}. The real attribute corresponds to $a$, the imaginary attribute corresponds to $b$.\\
Add a function {\code{DisplayImaginaryNumber()}} that prints your number in the following
format: {\code{a + i * b}} where {\code{a}} and {\code{b}} are place holder.\\
Add a function {\code{AddImaginaryNumbers()}} that adds two {\code{Complex}} objects together. To add
two Complex objects together do:
\begin{center}
	$(a + i \cdot b) + (c + i \cdot d) = (a + c) + i \cdot (b + d)$
\end{center}
Add a function {\code{SubtractImaginaryNumbers()}} that subtracts two {\code{Complex}} objects from each other. To subtract two {\code{Complex}} objects do:
\begin{center}
	$(a + i \cdot b) - (c + i \cdot d) = (a - c) + i \cdot (b - d)$
\end{center}
